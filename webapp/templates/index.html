Olá!

Para minerar as notícias, basta navegar até o menu "Minerar", onde será retornado um arquivo JSON.

Se preferir visualizar as informações mineradas, navegue até o menu "Visualizar dados minerados".

Para realizar o reconhecimento de entidades nas notícias mineradas, navegue até o menu "Processar, onde será retornado um arquivo JSON.

Se preferir visualizar as entidades reconhecidas de cada notícia, navegue até o menu "Visualizar dados processados".
